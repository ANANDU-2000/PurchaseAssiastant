# HEXA — Super Admin (Vite + React + TS)

## Run

```bash
npm install
npm run dev
```

Default: http://localhost:5173 — sidebar routes match [docs/admin-panel.md](../docs/admin-panel.md).

Set `VITE_API_BASE_URL=http://localhost:8000` in `.env` when wiring `fetch` / admin JWT.

## Build

```bash
npm run build
```
