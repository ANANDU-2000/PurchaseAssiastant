# Compact purchase flow (trader mental model)

## Ordered steps (primary)

1. **Pick item** (search / last buy hint)
2. **Quantity + unit** (bags vs kg only when bag family)
3. **Purchase rate + selling rate** (same row)
4. **Tax OFF / ON**
5. **Preview** (short)
6. **Save**

## Deferred (Advanced)

Tax % fine-tune, freight, billty, delivered, discount, notes, HSN, legacy GST basis.

## Copy rules

- Verbs: “Save”, “Save & add more”, “Advanced”
- Avoid: normalized, taxable, pre-tax, basis, engine

## Success metric

Median time to add first line on a known item **without opening Advanced** < 20s on mid-range Android.
