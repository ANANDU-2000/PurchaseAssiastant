# Purchase card standard (History)

## Layout (LTR column)

1. **Supplier** — single line, ellipsis.
2. **Item headline** — one line name, or `N items` when multiple lines.
3. **Pack summary** — `purchaseHistoryPackSummary` (may wrap 2 lines, ellipsis).
4. **`humanId • purchaseDate`** — small neutral text.

**Right column:** formatted total (₹), compact payment chip (`_historyPaymentChipLabel`).

## Status chip labels

| Status | Chip |
|--------|------|
| paid | Paid |
| overdue | Overdue |
| draft | Draft |
| due_soon | Due soon |
| partially_paid | Partial |
| other active | Pending |

## Slidable actions

Unchanged: Edit, Paid, Share, Delete.
