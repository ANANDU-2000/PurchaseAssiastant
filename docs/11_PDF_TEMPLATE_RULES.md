# 11 — PDF Template Rules (Clean Wholesale Invoice)

## Must

- Compact typography, clean alignment
- No corrupted symbols
- No giant invoice title / giant hero numbers
- Table includes **Purchase rate** + **Selling rate**
- Package type clarity per row

## PDF table

`Item | Qty | Unit | P Rate | S Rate | Total`

Qty/unit examples:
- BAG: `100 BAG` (and optional `5000kg` as subtext)
- KG: `5000 KG`
- BOX: `100 BOX`
- TIN: `50 TIN`

## Never

- Duplicate totals sections
- Emoji
- Wide layouts that don’t print on mobile
