# Mobile Scan Layout (iPhone 16 Pro target)

## Rules
- Single scroll view (no nested scroll)
- Dense spacing (avoid “debug screen” whitespace)
- Sticky bottom action bar always visible
- Full-width image preview after selection
- Table-first detected items (no cards-only layout)

## Section order
Header → Upload card → Image preview → Progress → Supplier/Broker → Items table → Charges → Actions.
